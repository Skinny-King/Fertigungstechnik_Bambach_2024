    \item $x_s$ : \# Schichten 
    \item $x_d$ : Schichtdauer $\left[\text{h/d}\right]$
    \item $x_j$ : Jahresarbeitstage $\left[\text{d/j}\right]$
    \item $x_N$ : Nutzungsgrad
